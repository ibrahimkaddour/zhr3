Employee visa recruiter request and Expense calculation

Recruiter Manager can put the visa request, HR Officer approve it and Expense Manager calculate the expense

Employee Visa Recruiter Request

Here recruiter manager can put individual/family employee visa request
There are two types for visa:
  1. Individual
  2. Family

There are different visa type:
  1. Business Visit Visa
  2. Work Visit Visa
  3. New Work Visa

And also add details based on purpose of visa and visa type Like, Employee Information, Request By, Departure Date, Return Date. Then submit the request.


In Process Confirmation


HR Officer can process any visa request or they can accept/reject the request also add person name who's handle this request.



Received Visa


HR Officer received visa or refuse visa after add visa number, approve date from and approve date to.


Generate Visa Expense


Expense Manager will add expense (company & employee contribution) details & Generate expense.


Expense Process


Expense Manager will do expense approval process.
